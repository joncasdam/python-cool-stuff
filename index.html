SimpleHTTPServer

Want to know how your site looks on the server? Just launch the python built-in SimpleHTTPServer module:

				$ python -m SimpleHTTPServer
$ Serving HTTP on 0.0.0.0 port 8000 ...

defaultdict

With defauldict you'll never get a KeyError. Instead, it will always return the factory function you passed in:

				>>> from collections import defaultdict
>>> s = 'mississippi'
>>> d = defaultdict(int)
>>> for k in s:
...     d[k] += 1
...
>>> d.items()
[('i', 4), ('p', 2), ('s', 4), ('m', 1)]
